# Autism Awareness Website

A React-based website promoting autism awareness, featuring smooth animations with **AOS**, dynamic state management using `useContext`, and Calendly integration for easy appointment scheduling. Designed to provide valuable resources and foster a supportive community for individuals and families affected by autism.
